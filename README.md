# 고사성어 이야기 — 한문 비주얼 노벨

한자 고사성어를 이야기(비주얼 노벨) 형식으로 배우는 웹 페이지입니다.
별도의 빌드나 서버 없이 `index.html` 하나로 동작합니다.

## 구성

```
story/
├── index.html            # 비주얼 노벨 본체 (열면 바로 실행)
├── images/               # 이야기 삽화 (16:9)
│   ├── cover.png         # 표지
│   ├── saeongjima_1.png  # 새옹지마 장면 1
│   └── saeongjima_2.png  # 새옹지마 장면 2
└── 이미지_프롬프트_팩.md   # 삽화 생성용 AI 이미지 프롬프트 모음
```

## 실행 방법

`index.html`을 브라우저에서 열면 됩니다.

GitHub Pages를 켜면 웹에서 바로 볼 수 있습니다:
저장소 **Settings → Pages → Branch: main** 선택 후 저장.

## 삽화 추가 방법

1. `이미지_프롬프트_팩.md`의 장면 프롬프트 + 공통 화풍 블록으로 이미지를 생성합니다.
2. 문서에 적힌 파일명 그대로 `images/` 폴더에 저장하면 사이트에 자동으로 반영됩니다.
   (파일명 규칙: `images/<고사성어id>_<장면번호>.png`, 예: `saeongjima_1.png`)
